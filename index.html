Privacy posture

Built around local ownership from the start.

Cove Mail is designed around account-scoped settings, local cached metadata, secure credential storage, and explicit boundaries for automation. Future providers can be added without changing the first principle: your mail client should feel like it belongs to you.

Keychain credentials Passwords are stored through macOS Keychain, not plain local files.
Metadata-first tools Automation starts with structured message metadata and safe snippets.
Permissioned writes Drafting, sending, deleting, and folder changes stay reviewable.
